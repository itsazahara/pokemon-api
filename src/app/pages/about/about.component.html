Sobre el proyecto

Este proyecto fue desarrollado como parte del curso de Angular, con el objetivo de aprender y practicar el consumo de APIs REST y la creación de interfaces dinámicas en una aplicación web.

Desarrollador/a

Azahara Romero Schulte

Desarrolladora de Aplicaciones Web

Feedback

Pros:

  • Aprendizaje de Angular, una potente librería para construir aplicaciones web dinámicas.
  • Consumo de APIs RESTful, permitiendo una integración fluida con datos externos.
  • Mejora en la comprensión de conceptos clave como servicios, rutas y componentes.

Contras:

  • Desafíos con la paginación y gestión de grandes volúmenes de datos.
  • Algunos problemas al hacer llamadas a la API, lo que requería depuración en tiempo real.
  • Curva de aprendizaje con la integración de funcionalidades avanzadas de Angular.

Elementos Desafiantes

  • Trabajo con rutas dinámicas y el paso de parámetros entre vistas fue un reto.
  • Personalización de la interfaz para mejorar la experiencia de usuario con Angular Material.
  • Gestionar el estado y flujo de datos de manera eficiente en componentes complejos.

Conclusión

Este proyecto ha sido una excelente oportunidad para aplicar mis conocimientos de Angular y consumo de APIs en un contexto real. Si bien hubo algunos desafíos, la experiencia ha sido muy positiva, y ha mejorado mi habilidad para trabajar con tecnologías de frontend modernas, al punto de querer aplicar y usar lo aprendido en mi trabajo de fin de grado.
